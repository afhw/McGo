# 个性化、主页、音乐与联机

## 主题与背景

进入“管理 -> 环境”，可以切换 dark、light、auto 主题，并选择本地背景图片。

## 自定义主页

主页内容支持本地 `.txt` / `.md` 文件。高级需求下可以启用“允许联网主页纯文本”，然后填写 `http(s)` 地址。联网内容只按纯文本展示，不执行脚本。

## 背景音乐

进入“管理 -> 环境 -> 个性化与功能”：

- 选择本地音频文件。
- 设置音量。
- 勾选“启用背景音乐”后点击“播放/应用”。
- 可设置游戏启动后自动暂停音乐。

## 页面隐藏

可以隐藏下载页或管理页，降低普通使用时的导航复杂度。隐藏管理页后需要编辑 `launcher_config.ini` 中 `[FEATURES] show_manage=True` 才能恢复。

## 联机入口

联机已独立为主导航页面。进入“联机”可以维护常用服务器地址、检测当前 NAT 类型，并启动 McGo P2P 隧道。

服务器列表每行一个地址，例如：

```text
mc.example.com:25565 | 生存服
play.example.net | 小游戏
```

点击“复制首个服务器”会复制第一行 `|` 前的地址。

NAT 检测使用 STUN 探测 UDP 公网映射；如果结果显示对称 NAT、地址相关映射或多出口 NAT，建议优先使用中继联机。

同一页面也提供 McGo P2P 隧道。默认中继为 `flyliq.cn:10721`，房主填写 Minecraft 局域网端口并启动“作为房主”，加入者填写同一个房间后启动“作为加入者”，再在 Minecraft 中连接 `127.0.0.1:本地监听端口`。服务端部署见 [P2P 联机与中继部署](p2p.md)。
