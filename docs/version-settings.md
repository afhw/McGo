# 版本设置与启动参数

进入“启动 -> 版本设置”管理当前版本。

## 个性化

- 显示名称：给版本起别名。
- 版本图标：在版本列表中显示简单标识。
- 收藏当前版本：收藏后会优先排序，并出现在“收藏”分类中。
- 隐藏当前版本：从普通分类中移除，只在“隐藏”分类中显示。

“启动 -> 选择版本”提供分类选择框和版本列表。分类规则：

- 全部版本：显示未隐藏版本。
- 收藏：显示已收藏且未隐藏版本。
- 原版 / 仅 OptiFine / 可安装 Mod：按版本类型筛选，隐藏版本不会出现。
- 隐藏：只显示被标记隐藏的版本。

## 启动设置

- 最小内存：写入 `-Xms`，`0` 表示不设置。
- 最大内存：写入 `-Xmx`，`0` 时默认使用 2048 MB。
- 窗口宽度/高度：设置 Minecraft 启动分辨率。
- GC 策略：支持 G1GC、ZGC、Shenandoah 或默认。
- 额外 JVM 参数：追加到 Java 参数中。
- 额外游戏参数：追加到 Minecraft 游戏参数中。
- 启动前命令：启动游戏前在运行目录执行。
- 自定义运行目录：覆盖默认 `.minecraft` 或隔离目录。
- 资源隔离：让当前版本使用 `versions/<版本名>/` 作为运行目录。

## 快捷入口

版本设置页可以直接打开版本文件夹、存档、Mod、资源包、光影和截图目录，也可以导出启动脚本或整合包。
