# 快速开始与界面概览

## 安装依赖

```bash
python -m pip install -r requirements.txt
```

## 启动

```bash
python main.py
```

首次启动后建议按这个顺序操作：

1. 进入“管理 -> 账号”，添加离线、Microsoft 或外置登录账号。
2. 进入“管理 -> 环境”，确认游戏目录和 Java 路径。
3. 进入“下载”，刷新远程版本并下载 Minecraft。
4. 回到“启动”，选择账号和本地版本后启动游戏。

## 页面说明

- 首页：显示账号、Java、本地版本、远程版本和自定义主页内容。
- 启动：选择账号、版本、版本分类，并管理当前版本设置。
- 下载：下载原版、安装 Fabric/Forge/NeoForge/OptiFine/Fabric API、导入整合包、搜索资源。
- 管理：账号、环境、个性化、服务器入口、日志和帮助。
